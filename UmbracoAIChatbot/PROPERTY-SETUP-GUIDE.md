## Quick Copy-Paste Reference for Umbraco UI

### Tab 1: Hero Section

**Property 1:**
- Name: `Hero Title`
- Alias: `heroTitle`
- Data Type: **Textstring** (0cc0eba1-9960-42c9-bf9b-60e150b429ae)
- Description: Main headline for the hero section (HTML allowed for span tags)

- Name: `Hero Subtitle`

## üöÄ Steps to Add Properties

1. Go to **Settings** ‚Üí **Document Types** ‚Üí **BPO Homepage**
2. Click **Design** tab
3. Click **+ Add tab** ‚Üí Enter "Hero Section"
4. Inside "Hero Section" tab, click **+ Add property**
5. Fill in Property 1 details from above
6. Click **Submit**
7. Repeat for Property 2
8. Repeat steps 3-7 for "About Section" tab (Properties 3-9)
9. Repeat steps 3-7 for "Contact Section" tab (Properties 10-12)
10. Click green **Save** button at top right

---

## ‚úÖ After Adding Properties

1. Go to **Content** tab
2. Click on **BPO Homepage** document
3. You'll see 3 tabs with all the fields!
4. Fill in the content (or leave empty to use template defaults)
5. Click **Save and Publish**

Your website is now fully editable from the backoffice! üéâ
